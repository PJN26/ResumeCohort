# Resume Using HTML

A clean and professional HTML-based resume for Pujan Parekh, showcasing skills, experience, projects, and education.

## Overview

This project is a simple, static HTML resume built without any external CSS frameworks or JavaScript. It demonstrates fundamental HTML structure and semantic markup to present professional information in an organized format.

## Features

- **Personal Information**: Name, tagline, and social media links (LinkedIn, GitHub)
- **About Me**: Brief introduction and current academic status
- **Coding Profiles**: Link to LeetCode profile
- **Skills**: Technical skills including web development, programming languages, and databases
- **Experience**: Professional experience displayed in a structured table format
- **Projects**: Detailed descriptions of key projects:
  - Hotel Apartment Booking Website (Node.js, Express.js, MongoDB)
  - Document Verification System (Python, OCR, Flask, OpenCV)
- **Education**: Academic qualifications in table format
- **Contact Information**: Email, phone, and location details

## Technologies Used

- HTML5
- Semantic HTML elements
- Basic HTML tables for structured data

## Structure

The resume is organized into clearly defined sections:
- Header with name and professional links
- About section
- Coding profiles
- Skills list
- Experience table
- Projects with detailed descriptions
- Education table
- Contact information
- Footer with copyright

## How to View

1. Clone or download this repository
2. Open `Resume.html` in any modern web browser
3. The resume will display in a clean, readable format

```bash
# Open in default browser (Windows)
start Resume.html
```
